Arbetsprocess

  1. Till att börja med tittade jag om på föreläsningen för att kunna följa med i hur mycket av uppgiften ska skapas. Behövde också se en YouTube-video för att fräscha upp minnet på hur man får in ett repository till VSC.
  2. Sedan initierade jag projektet genom att skriva npm init i terminalen.
  3. Nu kommer själva nedladdningen av parcel, med kommandot npm install parcel --save-dev .
  4. Efter det var det dags att länka in CSS och JS på det sättet dokumentationen bekriver det. Kan ha skapat dev-branchen här för att inte utveckla allt i huvudgrenen.
  5. Sedan byggde jag upp projektet med en README-fil, .gitignore (puttade in package-lock.json, senare också andra filer) och kod-filerna (HTML, CSS, JS), men dem fick hamna i sin egna src-mapp.
  6. Började nu skriva lite HTML, CSS och en liten JS-console.log som är ställföreträdare till senare kod. Använder npm run start för att kunna se en live server i webbläsaren.
  7. I dist-mappen finns den minimerade koden från de tre språken. Jag tog också och minimerade samt omvandlade bilderna direkt i HTML:n genom att bland annat lägga till följande i koden: ?as=webp&width=250
